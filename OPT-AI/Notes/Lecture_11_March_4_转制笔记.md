## 第 ① 页：Convergence Results for Gradient Descent

- Let f: ℝⁿ → ℝ be convex with global minimum x\*. We minimize f using gradient descent with constant step size γ.

$$x^{k+1} = x^k - \gamma \cdot g^k, \quad \text{where } g^k = \nabla f(x^k)$$

**Preliminary Analysis:**

- We did a preliminary analysis to bound the "error" in gradient descent cumulative over our K iterations:

$$\sum_{k=0}^{K-1} \left( f(x^k) - f(x^*) \right) \leq \frac{\gamma}{2} \sum_{k=0}^{K-1} \|g^k\|^2 + \frac{1}{2\gamma} \|x^0 - x^*\|^2$$

---

## 第 ② 页：收敛定理（一般凸函数情形）

**Thm:** Let f: ℝⁿ → ℝ convex with global minimum x\* and ‖∇f(x)‖ ≤ B. Assume that ‖x⁰ − x\*‖ ≤ R and we use gradient descent with constant step size

$$\gamma = \frac{R}{B\sqrt{K}}$$

Then, the iterates of gradient descent satisfy:

$$\frac{1}{K} \sum_{k=0}^{K-1} \left( f(x^k) - f(x^*) \right) \leq \frac{RB}{\sqrt{K}}$$

- By the theorem:

$$\min_{k=0,\ldots,K-1} f(x^k) - f(x^*) \leq \frac{1}{K} \sum_{k=0}^{K-1} \left( f(x^k) - f(x^*) \right) \leq \frac{RB}{\sqrt{K}}$$

- If we run gradient descent for K iterations, the best solution along the way will yield an objective value not deviating from optimal obj. value by at most RB/√K.

---

## 第 ③ 页：迭代复杂度（一般凸函数情形）

- Asking a similar question conversely, let's say we want to get a solution with error ε. How many iterations should we run gradient descent for?

$$\text{Set } \varepsilon = \frac{RB}{\sqrt{K}} \quad \Longrightarrow \quad K = \left( \frac{RB}{\varepsilon} \right)^2$$

- If ε = 0.1 , K = Const. · 100
- If ε = 0.001 , K = Const. · 1000000

The # of iterations increase fast as we want more precision.

## 第 ④ 页：Convergence of Gradient Descent for Smooth Convex Functions

**Definition:** Let f: ℝⁿ → ℝ be a convex function. We say that f is **smooth** with parameter L if

$$f(y) \leq f(x) + \nabla f(x)^\top (y - x) + \frac{L}{2} \|y - x\|^2 \quad \text{for all } x, y \in \mathbb{R}^n$$

> **[图示说明]** 坐标图，横轴为 y，纵轴为函数值。图中绘制了三条曲线：
> 1. 实际函数曲线 f(y)（蓝色，U形凸函数）
> 2. 二次上界曲线 f(x) + ∇f(x)ᵀ(y−x) + (L/2)‖y−x‖²（位于 f(y) 之上，红色标注）
> 3. 线性下界（切线）f(x) + ∇f(x)ᵀ(y−x)（位于 f(y) 之下，红色直线）
>
> 三条曲线在点 x 处相切。图示目的：直观说明光滑凸函数被夹在线性近似（下界）和二次近似（上界）之间。

Remember also for convex f, we have:

$$f(y) \geq f(x) + \nabla f(x)^\top (y - x)$$

- Intuitively, a smooth convex function lies below a local quadratic approximation, above a local linear approximation.

## 第 ⑤ 页：两个预备结论

We need two preliminary results for analyzing gradient descent for convex smooth functions.

**<u>Result 1:</u>** Let f: ℝⁿ → ℝ be smooth and convex and x⁰, x¹, x², ... are generated by gradient descent with constant step size γ

$$x^{k+1} = x^k - \gamma g^k, \quad g^k = \nabla f(x^k)$$

Then,

$$f(x^{k+1}) \leq f(x^k) - \gamma \left(1 - \frac{\gamma L}{2}\right) \|g^k\|^2$$

**Pf:** Because f is smooth with parameter L,

$$f(x^{k+1}) \leq f(x^k) + \nabla f(x^k)^\top (x^{k+1} - x^k) + \frac{L}{2} \|x^{k+1} - x^k\|^2$$

其中 ∇f(xᵏ) = gᵏ，xᵏ⁺¹ − xᵏ = −γgᵏ，‖xᵏ⁺¹ − xᵏ‖ = γ‖gᵏ‖，代入得：

$$= f(x^k) - \gamma (g^k)^\top g^k + \frac{L}{2} \gamma^2 \|g^k\|^2$$

$$= f(x^k) - \gamma \left(1 - \frac{L\gamma}{2}\right) \|g^k\|^2 \quad \text{done!}$$

- Thus, with γ = 1/L:

$$f(x^{k+1}) \leq f(x^k) - \frac{1}{2L} \|g^k\|^2$$

## 第 ⑥ 页：预备结论 2

**<u>Result 2:</u>** This is our earlier preliminary analysis:

$$\sum_{k=0}^{K-1} \left( f(x^k) - f(x^*) \right) \leq \frac{\gamma}{2} \sum_{k=0}^{K-1} \|g^k\|^2 + \frac{1}{2\gamma} \|x^0 - x^*\|^2$$

## 第 ⑦ 页：光滑凸函数的梯度下降收敛定理

**<u>Theorem:</u>** Let f: ℝⁿ → ℝ be a smooth convex function with parameter L having global minimum x\*.

Assume that ‖x⁰ − x\*‖ ≤ R.

Choosing step size γ = 1/L, the iterates of gradient descent algorithm satisfy:

$$f(x^K) - f(x^*) \leq \frac{L}{2K} \cdot R^2$$

**Pf:** By result 1 above,

$$\frac{1}{2L} \|g^k\|^2 \leq f(x^k) - f(x^{k+1})$$

Add this inequality over k = 0, ..., K−1:

$$\frac{1}{2L} \sum_{k=0}^{K-1} \|g^k\|^2 \leq f(x^0) - f(x^K) \quad (\text{use telescoping sums}) \quad (\ast)$$

By result 2 with step size γ = 1/L:

$$\sum_{k=0}^{K-1} \left( f(x^k) - f(x^*) \right) \leq \frac{1}{2L} \sum_{k=0}^{K-1} \|g^k\|^2 + \frac{L}{2} \|x^0 - x^*\|^2$$

$$\leq f(x^0) - f(x^K) + \frac{L}{2} R^2 \quad \Longrightarrow$$

*(由 (\*) 式代入)*

---

## 第 ⑧ 页：定理证明（续）

$$f(x^0) - f(x^*) + \sum_{k=1}^{K-1} \left( f(x^k) - f(x^*) \right) - f(x^0) + f(x^K) \leq \frac{L}{2} R^2 \quad \Longrightarrow$$

$$\sum_{k=1}^{K} \left( f(x^k) - f(x^*) \right) \leq \frac{L}{2} R^2 \quad \Longrightarrow$$

$$\frac{1}{K} \sum_{k=1}^{K} \left( f(x^k) - f(x^*) \right) \leq \frac{L}{2K} R^2$$

- By result 1:

  f(x¹) ≥ f(x²) ≥ f(x³) ≥ ... ≥ f(xᴷ)

  so

  f(x¹) − f(x\*) ≥ f(x²) − f(x\*) ≥ f(x³) − f(x\*) ≥ ... ≥ f(xᴷ) − f(x\*)

- We get:

$$f(x^K) - f(x^*) \leq \frac{1}{K} \sum_{k=1}^{K} \left( f(x^k) - f(x^*) \right) \leq \frac{L}{2K} R^2 \quad \text{Done!}$$

## 第 ⑨ 页：迭代复杂度（光滑凸函数情形）与渐近收敛

- How many iterations to run gradient descent for so that the iterate of the last iteration gives a function value that deviates from optimal by ε?

$$\text{Set } \varepsilon = \frac{L}{2K} R^2 \quad \Longrightarrow \quad K = \frac{LR^2}{2\varepsilon}$$

- If ε = 0.1 , then K = Const. · 10
- If ε = 0.001 , then K = Const. · 1000

- Also, we have:

$$0 \leq f(x^K) - f(x^*) \leq \frac{L}{2K} R^2$$

- Thus,

$$0 \leq \lim_{K \to \infty} \left( f(x^K) - f(x^*) \right) \leq \lim_{K \to \infty} \frac{L}{2K} R^2 = 0$$

- So:

$$\lim_{K \to \infty} \left( f(x^K) - f(x^*) \right) = 0 \quad \Longrightarrow \quad \lim_{K \to \infty} f(x^K) = f(x^*)$$

So if we run gradient descent long, the last iterate will be arbitrarily close to optimal.
