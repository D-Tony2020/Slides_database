## 讲解大纲

本讲从 Lecture 1 的逻辑回归自然延续，完成二分类的对数似然推导后，扩展到多类分类（softmax），最终引入深度学习——展示神经网络训练本质上也是一个优化问题。

**整体脉络：**

1. **逻辑回归回顾与对数似然推导**（第①-②页）：复习二分类逻辑回归的设定，详细推导似然函数和对数似然，得到最终的最大化优化问题。
2. **多类逻辑回归 / Softmax**（第③-⑤页）：将二分类推广到 K 类，引入 softmax 函数，推导多类对数似然的紧凑表达式。
3. **数值示例**（第⑥页）：用收入和教育年限预测类别的具体例子，让公式落地。
4. **深度学习**（第⑦-⑧页）：引入多层神经网络，展示前向传播结构，说明深度学习就是用神经网络提取特征后做多类逻辑回归。

**核心线索：** 从二分类 → 多分类 → 深度学习，模型越来越复杂，但优化框架始终一致：最大化对数似然（或等价地最小化交叉熵损失）。

## 逐页精讲

---

### 第 ① 页：Logistic Regression（回顾）

**原文翻译：**

逻辑回归。考虑一个两类分类问题，数据集 {(xⱼ, yⱼ) : j=1,...,M}，其中 xⱼ ∈ ℝⁿ，yⱼ ∈ {+1, −1}。我们的预测函数形如 ϕ(xⱼ, w) = 1/(1 + eʷᵀˣʲ)，可解释为特征向量 xⱼ 的数据点属于类别 +1 的概率。1 − ϕ(xⱼ, w) = eʷᵀˣʲ/(1 + eʷᵀˣʲ) 则是属于类别 −1 的概率。

**讲解：**

这一页是对 Lecture 1 末尾内容的回顾，确保逻辑回归的基本设定在学生脑中清晰。教授再次强调了三个关键点：（1）分类标签是 {+1, −1}，不是 {0, 1}——这个编码选择会影响后续似然函数的具体形式；（2）ϕ(x, w) = 1/(1+eʷᵀˣ) 输出的是属于 +1 类的概率，值域在 (0, 1) 之间；（3）1−ϕ 自动给出另一类的概率，保证了概率的归一化。这个回顾为下一页的似然推导做好铺垫——后续的推导需要根据每个数据点的标签选择使用 ϕ 还是 1−ϕ 作为其似然贡献。

---

### 第 ② 页：对数似然推导

**原文翻译：**

表格列出每个数据点的特征向量、标签和对应似然。数据集的似然函数为所有单点似然的乘积 L(w)。取对数后 ln L(w) 可化简为两项之和：−∑ln(1+eʷᵀˣʲ) + ∑𝟙(yⱼ=−1)·wᵀxⱼ。最终的优化问题为最大化这个对数似然。

**讲解：**

这一页是本讲的数学核心之一，教授一步步展示了从似然函数到优化问题的完整推导过程。

表格使用了一个具体的标签序列（+1, −1, −1, ..., +1）来展示规律：标签为 +1 的数据点贡献 1/(1+eʷᵀˣ)，标签为 −1 的贡献 eʷᵀˣ/(1+eʷᵀˣ)。取对数后，乘积变为求和。对于 +1 类的点，ln[1/(1+eʷᵀˣ)] = −ln(1+eʷᵀˣ)；对于 −1 类的点，ln[eʷᵀˣ/(1+eʷᵀˣ)] = wᵀx − ln(1+eʷᵀˣ)。

观察到每个数据点都贡献了 −ln(1+eʷᵀˣʲ) 这一项，而 wᵀxⱼ 项只在 yⱼ = −1 时出现。因此用指示函数 𝟙(yⱼ=−1) 可以写出统一的紧凑表达式。这种从"逐项展开"到"统一公式"的推导方式在统计学中非常典型，也体现了教授的教学风格——先让学生看到具体模式，再抽象概括。

最终的优化问题是一个**无约束的凹函数最大化**（等价于凸函数最小化），可以用梯度下降等方法求解。

### 第 ③ 页：Multi-Class Logistic Regression

**原文翻译：**

多类逻辑回归。考虑多类分类问题，数据集 {(xⱼ, yⱼ) : j=1,...,M}，其中 xⱼ ∈ ℝⁿ，yⱼ ∈ {1,...,K}。预测函数形如 ϕₖ(xⱼ, w₁,...,wₖ) = eʷₖᵀˣʲ / ∑ₗeʷₗᵀˣʲ，即特征向量 xⱼ 的数据点属于类别 k 的概率。使用最大似然估计来估计参数 (w₁,...,wₖ) ∈ ℝⁿˣᴷ。

**讲解：**

这是从二分类到多分类的关键推广。当类别数 K > 2 时，sigmoid 函数不再适用——它只能输出一个概率值，而多类问题需要 K 个概率且和为 1。解决方案是 **softmax 函数**：对每个类别 k，计算 eʷₖᵀˣ，然后除以所有类别的 eʷₗᵀˣ 之和，得到归一化的概率分布。

softmax 的几个重要性质：（1）所有输出都是正数（指数函数保证）；（2）所有输出之和为 1（除以总和保证）；（3）wₖᵀx 越大，第 k 类的概率越高——每个参数向量 wₖ 本质上定义了一个"模板"，与输入的匹配程度（内积大小）决定分类概率。

注意参数空间从二分类的 ℝⁿ 扩展到了 ℝⁿˣᴷ——每个类别需要一个独立的参数向量。这使得参数数量增加了 K 倍，优化问题的维度也相应增大。当 K = 2 时，softmax 退化为 sigmoid（可以验证）。

### 第 ④ 页：多类逻辑回归的似然与对数似然

**原文翻译：**

表格展示每个数据点的特征、标签（具体数值如 3, 1, 4, ..., 3）和对应的 softmax 似然。似然函数为乘积形式。对数似然展开后，化简为两项：∑ⱼ∑ₖ 𝟙(yⱼ=k)wₖᵀxⱼ − ∑ⱼ ln(∑ₗ eʷₗᵀˣʲ)。

**讲解：**

这一页是第②页推导的多类推广版本，教授用完全相同的"先展开具体例子，再统一抽象"的方法论来推导。

表格中的标签是具体的数字（3, 1, 4, ..., 3），让学生能够直观地看到：数据点 1 的标签是 3，所以它的似然贡献就是 softmax 中第 3 类的概率 eʷ₃ᵀˣ¹/∑eʷₗᵀˣ¹。

取对数后的化简是本页的关键推导。每个似然项 ln(eʷᵧⱼᵀˣʲ/∑eʷₗᵀˣʲ) = wᵧⱼᵀxⱼ − ln(∑eʷₗᵀˣʲ)。第一项就是"正确类别的得分"（用指示函数 𝟙(yⱼ=k) 来选出正确类别 k 的 wₖᵀxⱼ），第二项是 log-sum-exp——对所有类别得分的"软最大值"。最终的对数似然是所有数据点上这两项的求和。

对数似然的结构非常清晰：**最大化 = 提高正确类别的得分 − 降低所有类别得分的整体水平**。这就是交叉熵损失的本质——它驱动模型将概率集中到正确类别上。

### 第 ⑤ 页：多类逻辑回归最终优化问题

**原文翻译：**

max_{(w₁,...,wₖ)∈ℝⁿˣᴷ} { ∑ⱼ∑ₖ 𝟙(yⱼ=k) wₖᵀxⱼ − ∑ⱼ ln(∑ₗ eʷₗᵀˣʲ) }

**讲解：**

这一页仅展示了最终的优化公式，教授将其单独列出是为了强调：这就是多类逻辑回归的**核心优化问题**。所有前面页的推导都汇聚于此。

这个优化问题有几个重要特性：（1）它是**凹函数的最大化**（等价于凸函数最小化），因为第一项是 w 的线性函数，而 log-sum-exp 是凸函数，取负后为凹，因此整体为凹。这意味着任何局部最优就是全局最优。（2）它是**无约束优化**——没有对参数的限制，搜索空间是整个 ℝⁿˣᴷ。（3）虽然有闭式梯度（log-sum-exp 的梯度有漂亮的形式），但没有闭式解——必须用迭代方法。

从优化的角度看，这是一个"很友好"的问题——凸、无约束、可微。但当数据量 M 或特征维度 n 或类别数 K 很大时，计算成本仍然很高——这就引出了后续课程中的随机梯度下降等方法。

### 第 ⑥ 页：数值示例

**原文翻译：**

每个数据点的特征为（收入，学校教育年限）。数据点 1：特征 (50K, 11)，标签 A；数据点 2：特征 (150K, 16)，标签 B。参数向量 (wₐ, wᵦ) = ((wₐ,收入, wₐ,教育年限), (wᵦ,收入, wᵦ,教育年限)) = ((0.1, 0.5), (0.4, 0.1))。

**讲解：**

教授通过一个简洁的两类两特征示例让 softmax 公式具象化。两个特征是"收入"和"受教育年限"，两个类别是 A 和 B——可以想象为预测一个人属于某个消费群体还是另一个。

参数 wₐ = (0.1, 0.5) 意味着 A 类更看重教育年限（系数 0.5）而非收入（系数 0.1）；wᵦ = (0.4, 0.1) 则意味着 B 类更看重收入。这非常直观——不同类别对特征的"偏好"由参数编码。

代入具体数值：数据点 1（50K 收入, 11 年教育）的 A 类得分 = 0.1×50 + 0.5×11 = 10.5，B 类得分 = 0.4×50 + 0.1×11 = 21.1。由于 B 类得分更高，softmax 会给 B 类更高的概率——但实际标签是 A，说明当前参数还不够好，需要通过优化来调整。

这个例子帮助学生建立从"抽象公式"到"具体计算"的桥梁，也为理解优化目标提供了直觉：优化就是要调整 wₐ 和 wᵦ，使得每个数据点的正确类别得分尽可能高于其他类别。

### 第 ⑦ 页：Deep Learning

**原文翻译：**

深度学习。考虑多类分类问题，数据点 {(xⱼ, yⱼ)}，xⱼ ∈ ℝⁿ，yⱼ ∈ {1,2,...,K}。神经网络由 L 层组成，每层有参数 (Qˡ, gˡ) 和激活函数 σˡ。第一层输出 aⱼ¹ = σ₁(Q¹xⱼ + g¹)，第二层输出 aⱼ² = σ₂(Q²aⱼ¹ + g²)。激活函数包括 sigmoid σ(t) = 1/(1+eᵗ) 和 ReLU σ(t) = max{t, 0}。

**讲解：**

这一页是本讲的重要转折——从"浅层"线性模型跃升到"深层"神经网络。教授的展示方式非常系统：用一个流程图清晰地展示了深度网络的层级结构。

每一层都执行相同的基本操作：**线性变换 + 非线性激活**。第 l 层接收上一层的输出 aⱼˡ⁻¹（或原始输入 xⱼ），通过权重矩阵 Qˡ 做线性变换，加上偏置 gˡ，再通过激活函数 σˡ。教授特别标注了每个矩阵和向量的维度——Q¹ 是 d₁×n（将 n 维输入映射到 d₁ 维隐层），g¹ 是 d₁×1（偏置），输出 aⱼ¹ 是 d₁×1。维度的匹配是实现深度网络时最基本也最容易出错的地方。

激活函数的选择是关键设计决策。教授提到了两种经典选择：**sigmoid** 1/(1+eᵗ) 和 **ReLU** max{t, 0}。ReLU 是现代深度学习的默认选择——计算简单、不存在梯度消失问题。但从优化角度看，ReLU 在 t=0 处不可微，这给基于梯度的优化带来了理论上的挑战（实践中使用次梯度）。

---

### 第 ⑧ 页：深度网络的整体表达式

**原文翻译：**

给定神经网络的参数 (Q¹, g¹, Q², g², ..., Qᴸ, gᴸ)，对应输入 xⱼ 到第一层，从最后一层得到的输出为 σₗ(Qᴸ σₗ₋₁(... σ₂(Q² σ₁(Q¹xⱼ + g¹) + g²) ...) + gᴸ)。记为 z(xⱼ, Q¹, g¹, Q², g², ..., Qᴸ, gᴸ)。用 z 作为数据点 j 的特征，yⱼ 作为标签，来求解多类逻辑回归问题。

**讲解：**

这一页将上一页的逐层描述压缩成一个嵌套的数学表达式，揭示了深度网络的本质：它是一个**高度非线性的复合函数**。教授用花括号逐层标注——最内层是"第一层的输出"，层层嵌套直到"最后一层的输出"，让嵌套结构一目了然。

然后教授做了一个关键的概念统一：将整个网络的输出 z(xⱼ, ...) 视为数据点 xⱼ 的"深度特征"，然后在这个特征空间上做多类逻辑回归（softmax 分类）。这意味着深度学习 = 深度特征提取 + softmax 分类。前面学的 softmax 优化框架完全适用，只不过"特征"不再是原始输入，而是神经网络学到的表示。

从优化角度看，这带来了根本性的变化：虽然 softmax 本身是凸优化，但由于特征 z 是参数的非线性函数（多层嵌套），整个优化问题变成了**非凸**的——目标函数有多个局部最小值。这正是深度学习优化的核心难题，也是为什么第②页提到的"非线性优化软件可能失败"在这里尤为相关。理解这一点为后续学习随机梯度下降、学习率调度等深度学习优化技术奠定了认知基础。
