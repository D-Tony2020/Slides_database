# Lecture 12 — March 9, 2026 — 讲解笔记

## 讲解大纲

本讲引入凸集的概念，建立凸函数下水平集的凸性和约束优化可行域的凸性，然后转入投影梯度下降——将梯度下降推广到约束优化问题。

**整体脉络：**

1. **凸集的定义与直觉**（第①页）：连接集合中任意两点的线段完全在集合内部。
2. **凸集的交集性质**（第②页）：任意多个凸集的交集仍然是凸集。
3. **凸函数的下水平集是凸集**（第③页）：{x : f(x) ≤ 0} 是凸集。
4. **约束优化的可行域是凸集**（第④页）：凸约束的交集给出凸可行域。
5. **严格凸函数在凸集上的唯一最优解**（第⑤-⑥页）：严格凸+凸约束→唯一最优解。
6. **投影梯度下降的动机与定义**（第⑦-⑧页）：梯度下降步可能离开可行域，投影将其拉回。

**核心线索：** 凸集 → 凸约束 → 凸可行域 → 投影梯度下降——将无约束梯度下降自然推广到约束优化。

## 逐页精讲

---

### 第 ① 页：Convex Sets

**原文翻译：**

凸集定义：C ⊆ ℝⁿ 是凸的，如果对所有 x, y ∈ C 和 λ ∈ [0,1]，有 λx+(1−λ)y ∈ C。配图对比凸集和非凸集。直觉上，凸集中任意两点之间有"直接视线"。

**讲解：**

凸集的概念比凸函数更基本——凸函数的定义需要用到凸组合 λx+(1−λ)y 在定义域中，而凸集正是保证这一点的几何结构。

配图的直觉非常到位：左图的椭圆形集合，任取两点连线段都在集合内——"视线"畅通。右图的弯月形集合，某些点之间的连线穿出集合——"视线"被阻挡。

教授用"direct line of sight"这个比喻非常巧妙。在计算几何和机器人路径规划中，凸集的这个性质直接对应"任意两点之间可以直线到达而不撞墙"的概念。

常见凸集举例：半空间 {x : aᵀx ≤ b}、球 {x : ‖x−c‖ ≤ r}、超立方体 [a₁,b₁]×...×[aₙ,bₙ]、所有线性约束的交集（多面体）。

---

### 第 ② 页：凸集的交集性质

**原文翻译：**

如果 C₁, C₂, ..., Cₗ 都是凸集，则 C₁∩C₂∩...∩Cₗ 也是凸集。证明：x, y ∈ 交集意味着 x, y 同时在每个 Cᵢ 中；由各 Cᵢ 的凸性得凸组合在每个 Cᵢ 中；因此凸组合在交集中。

**讲解：**

这个性质看似简单，却是约束优化理论的基石。证明方法是"逐个集合检查"——只需验证凸组合属于每一个 Cᵢ，就能确保它属于交集。

Venn 图很好地展示了三个凸集的交集（红色区域）仍然是凸的。教授的红色旁注"the intersection of bunch of convex sets is still convex"是一句需要记住的关键结论。

重要注意：凸集的**并集**通常不是凸集！例如两个不相交的圆的并集不是凸的。这就是为什么约束优化中用"∩"（同时满足所有约束）而非"∪"。

### 第 ③ 页：凸函数的下水平集是凸集

**原文翻译：**

若 f 凸，则 C = {x : f(x) ≤ 0} 是凸集。证明：x, y ∈ C 意味着 f(x), f(y) ≤ 0。加权平均 λf(x)+(1−λ)f(y) ≤ 0。由凸性 f(λx+(1−λ)y) ≤ λf(x)+(1−λ)f(y) ≤ 0，故凸组合在 C 中。

**讲解：**

这个结论将**凸函数**和**凸集**连接起来。{x : f(x) ≤ c} 称为 f 的 c-下水平集（sublevel set）。对于凸函数，任何下水平集都是凸集。

证明的逻辑链非常清晰：
1. f(x) ≤ 0 且 f(y) ≤ 0（在集合中）
2. λf(x) + (1−λ)f(y) ≤ 0（非正数的凸组合仍非正）
3. f(λx+(1−λ)y) ≤ λf(x)+(1−λ)f(y)（凸性定义）
4. 因此 f(λx+(1−λ)y) ≤ 0（凸组合在集合中）

注意反向不成立：下水平集是凸集**不**意味着函数是凸的。例如 f(x) = x³ 不是凸函数，但 {x : x³ ≤ 0} = (−∞, 0] 是凸集。

### 第 ④ 页：约束优化问题的可行域是凸集

**原文翻译：**

约束优化问题 min f(x) s.t. gᵢ(x) ≤ 0（gᵢ 均为凸函数）的可行域 C = {x : g₁(x)≤0, ..., gₘ(x)≤0}。每个 Cᵢ = {x : gᵢ(x)≤0} 是凸集（由第③页），C = ∩Cᵢ 是凸集（由第②页）。

**讲解：**

这一页将前两页的结果组装成一个实用结论：**如果所有约束函数是凸的，则可行域是凸集。**

教授用了"乐高积木"式的论证：
- 积木1：凸函数的下水平集是凸集（第③页）
- 积木2：凸集的交集是凸集（第②页）
- 组装：每个约束的可行集是凸集，它们的交集（同时满足所有约束）也是凸集

这就是为什么"凸优化"如此重要：当目标函数凸、约束函数凸时，我们处理的是"凸函数在凸集上的最小化"——这类问题具有最好的理论性质。

### 第 ⑤ 页：Projected Gradient Descent

**原文翻译：**

严格凸的定义：不等式取严格小于号（且 x ≠ y, λ ∈ (0,1)）。定理：严格凸函数 g 在凸集 𝕏 上最小化有唯一最优解。

**讲解：**

严格凸与凸的区别在于：凸允许等号（如线性函数），严格凸要求不等号（排除线性函数和"平底"函数）。几何上，严格凸函数的图像严格弯曲——不包含直线段。

唯一性定理极其重要：在严格凸+凸约束的设定下，最优解是唯一的。这消除了"最优解不唯一"带来的各种麻烦——算法收敛到哪个最优解不再是问题，因为只有一个。

教授提到"proof on next page"——这在手写笔记中很常见，说明证明有些长度，需要单独一页展开。

### 第 ⑥ 页：严格凸函数在凸集上有唯一最优解的证明

**原文翻译：**

反证法：假设有两个不同的最优解 x\* 和 y\*，最优值为 z\*。由凸集性质 λx\*+(1−λ)y\* ∈ 𝕏。由严格凸性 g(λx\*+(1−λ)y\*) < λg(x\*)+(1−λ)g(y\*) = z\*。这意味着存在可行点的目标值严格小于 z\*，矛盾。

**讲解：**

这是一个经典的反证法，逻辑极其清晰：

1. **假设**存在两个不同的最优解 x\* ≠ y\*，目标值都是 z\*。
2. 取凸组合 z = λx\* + (1−λ)y\*。因为 x\*, y\* ∈ 𝕏 且 𝕏 凸，z ∈ 𝕏（**可行**）。
3. 因为 g 严格凸且 x\* ≠ y\*：g(z) < λg(x\*) + (1−λ)g(y\*) = λz\* + (1−λ)z\* = z\*。
4. 所以 z 可行且 g(z) < z\*——**严格优于**所谓的"最优值"。矛盾！

严格凸性在第3步中至关重要——如果只是凸（非严格），不等式变成 ≤，得到 g(z) ≤ z\*，这只说明 z 也是最优解，不构成矛盾。

### 第 ⑦ 页：投影梯度下降的动机

**原文翻译：**

在约束优化中，沿负梯度走可能离开可行域。配图：左图中 x−γ∇f(x) 在可行域外（✗）；右图中将 x−γ∇f(x) 投影回可行域表面。投影梯度下降：走一步梯度，如果出界就投影回去。

**讲解：**

这是从无约束梯度下降到约束梯度下降的自然过渡。问题很直观：梯度不"知道"约束的存在——它只关心目标函数的下降方向，完全可能指向可行域外。

两幅图展示了两种情况：
- **左图**：从可行域内部的 x 出发，梯度步 x − γ∇f(x) 落在可行域外部。如果直接接受这个点，约束就被违反了。
- **右图**：解决方案——将出界的点"投影"回可行域，找到可行域中距该点最近的点。

投影梯度下降的核心思想是"先走再拉"：先按照无约束梯度下降走一步（可能出界），然后投影回可行域。这比"只在可行域内走"更自然——梯度方向信息被充分利用，投影只修正"出界部分"。

---

### 第 ⑧ 页：投影算子与投影梯度下降算法

**原文翻译：**

投影定义：Π_𝕏(x) = argmin_{y∈𝕏} ‖y−x‖²——集合 𝕏 中距 x 最近的点。若 x ∈ 𝕏 则 Π_𝕏(x) = x。投影梯度下降：yᵏ⁺¹ = xᵏ − γ∇f(xᵏ)（梯度步），xᵏ⁺¹ = Π_𝕏(yᵏ⁺¹)（投影步）。

**讲解：**

投影算子 Π_𝕏 的定义优雅而精确：给定 𝕏 外的一个点 x，找到 𝕏 中距 x 最近的点。配图清楚展示：从 x 到圆形凸集作"垂线"，垂足就是投影。

由第⑤-⑥页的唯一性定理，当 𝕏 是凸集时，‖y−x‖² 是关于 y 的严格凸函数，所以投影 Π_𝕏(x) 是唯一的——投影算子良定义。

投影梯度下降的每一步包含两个子步骤：
1. **梯度步**：yᵏ⁺¹ = xᵏ − γ∇f(xᵏ)——普通梯度下降，可能出界。
2. **投影步**：xᵏ⁺¹ = Π_𝕏(yᵏ⁺¹)——将出界的点拉回可行域。

投影的计算难度取决于 𝕏 的形状：对于简单集合（半空间、球、box），投影有闭式解；对于复杂集合，投影本身就是一个优化问题。这也是投影梯度下降的主要局限——投影步可能很昂贵。

投影梯度下降的收敛分析与普通梯度下降类似，只需额外利用"投影不增加到最优解的距离"这个性质（投影的非扩张性）。
