# Lecture 10 — March 2, 2026 — 讲解笔记

## 讲解大纲

本讲是整门课的高潮之一——推导梯度下降在凸函数上的收敛率。这是课程从"算法描述"到"算法分析"的飞跃。

**整体脉络：**

1. **问题设定与诚实预期**（第①页）：常步长梯度下降无法精确收敛到最小值，但平均误差可以量化。
2. **弹跳示例**（第②页）：用 f(x) = (x−1)² 展示常步长下的振荡行为。
3. **预备分析**（第③-⑤页）：利用凸性一阶刻画、范数恒等式、伸缩和，建立累积误差的上界。
4. **上界的直觉解读**（第⑥页）：小步长 vs 大步长的 trade-off。
5. **收敛定理**（第⑦页）：在有界梯度假设下，最优步长为 R/(B√K)，平均误差以 O(1/√K) 衰减。
6. **定理证明**（第⑧页）：通过优化步长的选择完成证明。

**核心线索：** 凸性一阶刻画 → 内积上界 → 伸缩和 → 累积误差上界 → 最优步长 → O(1/√K) 收敛率。

## 逐页精讲

---

### 第 ① 页：Convergence of Gradient Descent for Convex Functions

**原文翻译：**

用常步长 γ 的梯度下降最小化凸函数 f: ℝⁿ → ℝ。生成序列 xᵏ⁺¹ = xᵏ − γ∇f(xᵏ)。常步长下无法建立精确收敛，最好的结果是在最小值附近弹跳。但即使如此，收敛结果仍有意义。

**讲解：**

教授在开头就设定了诚实的预期：常步长梯度下降**不会**收敛到最小值点。这与很多初学者的直觉相反——步长恒定意味着即使接近最小值，每步仍然迈出固定大小的步子，导致来回振荡。

但"弹跳"并不意味着"无用"。教授会证明：虽然单次迭代可能不好，但**平均**函数值会逐步接近最优值。这就像一个在目标附近来回跳动的弹球——虽然从不静止，但平均位置越来越接近目标。

为什么不用递减步长？递减步长（如 γᵏ = 1/k）可以实现精确收敛，但分析更复杂。常步长分析更简洁，且结论已经足够有用——在实践中（如深度学习训练），常步长被广泛使用。

---

### 第 ② 页：常步长梯度下降的弹跳示例

**原文翻译：**

最小化 f(x) = (x−1)²，初始点 x⁰ = 2，步长 γ = 1。计算：x¹ = 0, x² = 2, x³ = 0, ...永远在 0 和 2 之间弹跳，最小值 x* = 1 从不被访问。

**讲解：**

这个简单例子完美地展示了"弹跳"现象。f(x) = (x−1)² 的最小值在 x* = 1，但步长 γ = 1 时：
- 从 x = 2 出发，梯度 f'(2) = 2，走到 2 − 1×2 = 0
- 从 x = 0 出发，梯度 f'(0) = −2，走到 0 − 1×(−2) = 2
- 永远在 0 和 2 之间来回跳

但注意：f(0) = 1, f(2) = 1，而 f(x*) = f(1) = 0。所以平均误差 (f(0)+f(2))/2 − f(1) = 1，不会随迭代减少。这是因为步长 γ = 1 对这个问题来说太大了。如果选更小的步长（比如 γ = 0.1），弹跳幅度会小很多，平均误差也会小很多。

这引出了核心问题：**什么是"最优"步长？** 太小则进展太慢（初始距离项主导），太大则弹跳太严重（梯度项主导）。

---

### 第 ③ 页：Preliminary Analysis

**原文翻译：**

预备分析。回顾凸性一阶刻画。范数平方展开恒等式：‖x−y‖² = ‖x‖² + ‖y‖² − 2xᵀy。记 gᵏ = ∇f(xᵏ)，x* = argmin f(x)。

**讲解：**

教授建立了证明需要的所有工具：

**范数恒等式** ‖x−y‖² = ‖x‖² + ‖y‖² − 2xᵀy 看似简单，但在证明中起关键作用——它将**内积**（不好处理）转化为**范数的差**（好处理，可以伸缩）。这个技巧在凸优化分析中反复出现。

符号约定 gᵏ = ∇f(xᵏ) 使后续公式更简洁。x* 是真正的全局最小值点。目标是上界 f(xᵏ) − f(x*)——即第 k 步的"次优性"。

---

### 第 ④ 页：上界推导

**原文翻译：**

用凸性一阶刻画得 f(xᵏ)−f(x*) ≤ (gᵏ)ᵀ(xᵏ−x*)。由梯度下降更新得 gᵏ = (1/γ)(xᵏ−xᵏ⁺¹)。

**讲解：**

证明的第一步极其关键：凸性一阶刻画给出

$$f(x^*) \geq f(x^k) + \nabla f(x^k)^T(x^* - x^k)$$

移项得 f(xᵏ) − f(x*) ≤ ∇f(xᵏ)ᵀ(xᵏ − x*) = (gᵏ)ᵀ(xᵏ − x*)

这将函数值的差转化为**内积**。然后利用梯度下降的更新公式 xᵏ⁺¹ = xᵏ − γgᵏ 得到 gᵏ = (xᵏ − xᵏ⁺¹)/γ。

代入后，问题变成上界 (1/γ)(xᵏ − xᵏ⁺¹)ᵀ(xᵏ − x*)——一个纯粹的向量运算。

---

### 第 ⑤ 页：展开内积并利用伸缩和

**原文翻译：**

利用 ‖a−b‖² = ‖a‖²+‖b‖²−2aᵀb 将内积展开为范数的差。伸缩求和消去中间项。最终得到 Σ(f(xᵏ)−f(x*)) ≤ (γ/2)Σ‖gᵏ‖² + (1/(2γ))‖x⁰−x*‖²。

**讲解：**

这是全讲最精巧的代数操作。将 (xᵏ−xᵏ⁺¹)ᵀ(xᵏ−x*) 用恒等式 2aᵀb = ‖a‖²+‖b‖²−‖a−b‖² 展开（取 a = xᵏ−xᵏ⁺¹, b = xᵏ−x*）：

$$2(x^k-x^{k+1})^T(x^k-x^*) = \|x^k-x^{k+1}\|^2 + \|x^k-x^*\|^2 - \|x^{k+1}-x^*\|^2$$

右边出现了 ‖xᵏ−x*‖² − ‖xᵏ⁺¹−x*‖²——这是**伸缩和**（telescoping sum）的标志！对 k = 0, ..., K−1 求和后，中间项全部抵消，只剩下 ‖x⁰−x*‖² − ‖xᴷ−x*‖²。由于 ‖xᴷ−x*‖² ≥ 0，可以丢弃它得到上界。

最终的累积误差上界有两项：
- **(γ/2)Σ‖gᵏ‖²**：与梯度大小和步长成正比——步长大或梯度大导致"过冲"。
- **(1/(2γ))‖x⁰−x*‖²**：与初始距离成正比、与步长成反比——步长小导致"走不远"。

---

### 第 ⑥ 页：预备分析的总结与解读

**原文翻译：**

总结预备分析的三步策略。解读上界的含义：步长小时初始距离项主导（走不远），步长大时梯度项主导（过冲）。

**讲解：**

教授用直觉语言解释了数学上界的物理含义，这是优秀教学的体现。

**小步长极端：** γ → 0 时，(1/(2γ))‖x⁰−x*‖² → ∞。迭代点几乎不移动，停在初始点附近。误差由"你一开始有多远"决定。

**大步长极端：** γ → ∞ 时，(γ/2)Σ‖gᵏ‖² → ∞。每步迈出巨大的步子，即使梯度很小也会大幅移动。误差由"你过冲了多少"决定。

**最优步长**在两个极端之间取得平衡——既不太小也不太大。下一页将找到这个最优值。

---

### 第 ⑦ 页：Convergence of Gradient Descent Under Bounded Gradients

**原文翻译：**

假设 f 凸且 ‖∇f(x)‖ ≤ B（有界梯度）。定理：选步长 γ = R/(B√K)，K 步后平均误差 ≤ RB/√K。

**讲解：**

这是全课最重要的定理之一。结论说：**梯度下降的平均误差以 O(1/√K) 速率衰减。**

关键假设是**有界梯度** ‖∇f(x)‖ ≤ B——这在很多实际问题中成立（如正则化后的机器学习损失函数）。B 可以理解为"函数变化的最快速度"。

最优步长 γ = R/(B√K) 取决于三个量：
- R = ‖x⁰−x*‖：初始距离——离最优点越远需要步长越大。
- B：梯度上界——梯度越大需要步长越小（避免过冲）。
- K：总迭代次数——迭代越多步长应该越小。

注意步长依赖于 K——需要预先知道总迭代次数。这在实践中可以通过"倍增法"绕过。

收敛率 O(1/√K) 意味着：要使平均误差 ≤ ε，需要 K = O(R²B²/ε²) 次迭代。这是次线性的，但已经是常步长下的最优结果。

---

### 第 ⑧ 页：收敛定理的证明

**原文翻译：**

证明：将有界梯度代入预备分析的上界，得到累积误差 ≤ (γ/2)KB² + (1/(2γ))R²。对 γ 求导令其为零，解得 γ = R/(B√K)。代回得平均误差 ≤ RB/√K。

**讲解：**

证明简洁而优雅。核心步骤：

1. **代入有界梯度**：‖gᵏ‖ ≤ B，所以 Σ‖gᵏ‖² ≤ KB²。代入预备分析的上界。

2. **优化步长**：上界 H(γ) = (1/2)(KB²γ + R²/γ) 是 γ 的凸函数。求导 H'(γ) = (1/2)(KB² − R²/γ²) = 0，解得 γ* = R/(B√K)。

   这里的优化步长本身就是一个凸优化问题——教授用自己课程中教的工具来解决课程中的问题，形成完美的递归。

3. **代回**：H(γ*) = (1/2)(R/(B√K)·KB² + B√K/R·R²) = (1/2)(RB√K + RB√K) = RB√K。

4. **除以 K**：平均误差 ≤ RB√K / K = RB/√K。

整个证明只用到了凸性一阶刻画、范数恒等式和基本微积分。这种"用简单工具推导深刻结论"的风格是优化理论的美学特征。
