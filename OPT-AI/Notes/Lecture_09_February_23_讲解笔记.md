### 第 ① 页：复习与凸函数一阶刻画

**原文翻译：**

复习：o(t) 定义、Taylor 推论、凸函数定义。一阶刻画：f 凸当且仅当 f(y) ≥ f(x) + ∇f(x)ᵀ(y−x) 对所有 x,y 成立。配图展示切线在凸函数下方。

**讲解：**

教授用整整一页复习前几讲的三个核心结果，说明这些工具将在本讲中被反复使用。这种"层层叠加"的教学结构非常有效——每个新结论都建立在前面已证明的结论之上。

一阶刻画的直觉值得再次强调：凸函数"任何地方的切线都是全局下界"。这意味着：如果你站在凸函数曲面的某个点上，用切平面去估计远处的函数值，你永远不会**高估**——真实值总是大于等于切平面值。这个性质在其他类型的函数中是不成立的。

配图中绿色切线在凸函数曲线下方的图像，是本讲乃至整个课程中最重要的"心智模型"之一。

### 第 ② 页：一阶刻画定理的证明（⟹ 方向）

**原文翻译：**

凸函数和非凸函数的切线对比图。证明⟹方向：假设 f 凸，利用凸性定义得 f(x+λ(y−x)) ≤ (1−λ)f(x) + λf(y)，变形得 f(y) ≥ f(x) + [f(x+λ(y−x))−f(x)]/λ。

**讲解：**

两幅对比图极富教学价值：左图中凸函数的两条切线都在曲线下方，交叉形成"X"形；右图中非凸函数的切线在某些区域穿过曲线（不是全局下界）。

证明的起点是凸性定义：f((1−λ)x + λy) ≤ (1−λ)f(x) + λf(y)。这可以改写为：

$$f(y) \geq f(x) + \frac{f(x + \lambda(y-x)) - f(x)}{\lambda}$$

右边的分数 [f(x+λ(y−x)) − f(x)]/λ 有直觉含义：它是 f 沿 y−x 方向的"差分商"——当 λ → 0 时，它趋向方向导数 ∇f(x)ᵀ(y−x)。

关键洞察：对于凸函数，这个差分商**随 λ 递增**（分母变大但分子增长更快——这是凸性的另一种体现）。因此取 λ → 0 的极限给出最小值，即方向导数。

### 第 ③ 页：一阶刻画定理的证明（⟹ 方向续）

**原文翻译：**

将差分商用 Taylor 展开替换：[f(x+λ(y−x))−f(x)]/λ = ∇f(x)ᵀ(y−x) + o(λ‖y−x‖)/(λ‖y−x‖)·‖y−x‖。令 λ → 0，余项趋于零，得 f(y) ≥ f(x) + ∇f(x)ᵀ(y−x)。

**讲解：**

这一页将凸性定义和 Taylor 定理"焊接"在一起。核心推导步骤：

1. 从凸性得到 f(y) ≥ f(x) + [f(x+λ(y−x))−f(x)]/λ（对所有 λ ∈ (0,1)）。
2. 用 Taylor 展开替换分子：f(x+λ(y−x)) = f(x) + λ∇f(x)ᵀ(y−x) + o(λ‖y−x‖)。
3. 代入并化简得到 f(y) ≥ f(x) + ∇f(x)ᵀ(y−x) + [o(λ‖y−x‖)/(λ‖y−x‖)]·‖y−x‖。
4. 不等式对**所有** λ ∈ (0,1) 成立，令 λ → 0 时余项消失。

精妙之处在于：不等式的右边含有 λ，但左边 f(y) 不含 λ。这意味着不等式对所有 λ 都成立，因此对 λ 取极限也成立。极限消除了误差项，留下干净的一阶刻画。

### 第 ④ 页：一阶刻画定理的证明（⟸ 方向）

**原文翻译：**

反向证明：假设 f(y) ≥ f(x) + ∇f(x)ᵀ(y−x) 对所有 x,y 成立。取任意 a,b ∈ ℝⁿ 和 λ ∈ [0,1]。分别对 (a, λa+(1−λ)b) 和 (b, λa+(1−λ)b) 应用下界条件，加权求和后得到凸性不等式。

**讲解：**

反向证明极其优雅。策略是：要证 f(λa+(1−λ)b) ≤ λf(a) + (1−λ)f(b)，利用"切线是全局下界"这个假设。

在凸组合点 c = λa+(1−λ)b 处，假设条件给出两个不等式：
- f(a) ≥ f(c) + ∇f(c)ᵀ(a−c) = f(c) + (1−λ)∇f(c)ᵀ(a−b)
- f(b) ≥ f(c) + ∇f(c)ᵀ(b−c) = f(c) − λ∇f(c)ᵀ(a−b)

第一个乘以 λ，第二个乘以 (1−λ)，相加时梯度项恰好**抵消**：

λf(a) + (1−λ)f(b) ≥ λf(c) + (1−λ)f(c) = f(c) = f(λa+(1−λ)b)

这个梯度项抵消的"魔法"正是设计 λ 和 (1−λ) 权重的妙处——不同权重下梯度方向的贡献正好相反。

### 第 ⑤ 页：Minimizing Convex Functions

**原文翻译：**

全局最小值和局部最小值的正式定义。定理：f 凸且 x̂ 为局部最小值，则 x̂ 为全局最小值。证明：选 λ 使 λx̂+(1−λ)y 在 x̂ 的 ε 邻域内。

**讲解：**

这个定理是凸优化的基石：**凸函数没有"假谷底"**。对于非凸函数，梯度下降可能卡在局部最小值，无法到达全局最优。但对于凸函数，任何局部最小值都自动是全局最小值。

证明思路巧妙：要证 f(x̂) ≤ f(y) 对任意远处的 y，我们无法直接比较（x̂ 只是局部最小值，只能在 ε 邻域内比较）。解决方案：构造一个在 ε 邻域内的"桥接点" z = λx̂ + (1−λ)y。选择 λ 足够接近 1 使得 z 在 ε 邻域内。

---

### 第 ⑥ 页：局部最小值即全局最小值的证明完成

**原文翻译：**

λx̂+(1−λ)y 在 ε 邻域内，因此 f(x̂) ≤ f(λx̂+(1−λ)y)。又由凸性，f(λx̂+(1−λ)y) ≤ λf(x̂)+(1−λ)f(y)。合并得 f(x̂) ≤ (1−λ)f(y)+λf(x̂)，化简得 f(x̂) ≤ f(y)。第二个定理：若 f 凸且 ∇f(x̂)=0，则 x̂ 是全局最小值（由一阶刻画直接得出）。

**讲解：**

证明的逻辑链条如下：

1. z = λx̂+(1−λ)y 在 ε 邻域内 → f(x̂) ≤ f(z)（局部最小值性质）
2. f(z) ≤ λf(x̂) + (1−λ)f(y)（凸性）
3. 合并：f(x̂) ≤ λf(x̂) + (1−λ)f(y)
4. 移项：(1−λ)f(x̂) ≤ (1−λ)f(y)
5. 除以 (1−λ) > 0：f(x̂) ≤ f(y)

凸性在第2步中起了关键作用——它允许我们用"远处"的 y 的信息（通过加权平均）来建立不等式。

第二个定理更加简洁：如果 ∇f(x̂) = 0，由一阶刻画直接得 f(y) ≥ f(x̂) + 0 = f(x̂) 对所有 y 成立。这给出了凸函数全局最优的**充分条件**——梯度为零。

这两个定理为梯度下降在凸函数上的成功提供了理论保证：梯度下降收敛到驻点（梯度为零），而对凸函数，驻点就是全局最优。
