## 讲解大纲

本讲包含两个独立的重要主题：前半部分介绍约束优化的第二种方法——障碍函数法；后半部分开始为梯度下降的收敛率分析做数学准备——Taylor 定理。

**整体脉络：**

1. **障碍函数法**（第①-③页）：不同于惩罚函数法的"从外部逼近"，障碍函数法"从内部逼近"——在可行域边界建立无穷大的"屏障"，保证迭代点始终可行。
2. **障碍函数法的几何直觉与算法**（第②-③页）：三维几何图示和完整迭代算法。
3. **Taylor 定理预备：小 o 记号**（第④页）：定义 o(t) 记号，区分不同阶的增长率。
4. **Taylor 定理的陈述**（第⑤页）：多变量 Taylor 定理——梯度提供一阶线性近似。
5. **Taylor 定理的推论与证明**（第⑥-⑦页）：f(y) = f(x) + ∇f(x)ᵀ(y−x) + o(‖y−x‖) 的严格证明。

**核心线索：** 障碍函数法完成了约束优化的"粗糙方法"工具箱；Taylor 定理则是后续一切收敛率分析的数学基石。

## 逐页精讲

---

### 第 ① 页：Barrier Function Method for Constrained Optimization

**原文翻译：**

障碍函数法用于约束优化。只适用于仅含不等式约束的问题。当我们在可行域内部接近约束边界时，gᵢ(x) 是一个很小的负数，所以 −gᵢ(x) 是很小的正数，因此 1/(−gᵢ(x)) 是很大的正数。求解问题：min_x f(x) − Σμᵢ/gᵢ(x)，其中 μᵢ 是小的障碍乘子。

**讲解：**

障碍函数法（又称内点法）与上一讲的惩罚函数法形成鲜明对比。惩罚函数法允许迭代点在可行域外部，通过惩罚"拉回"可行域；障碍函数法则从一开始就保证迭代点在可行域**内部**，通过在边界设置"屏障"阻止点逃出可行域。

核心构造是 −μᵢ/gᵢ(x)：当 gᵢ(x) < 0（在可行域内部）时，−1/gᵢ(x) > 0 是有限的正数；当 gᵢ(x) → 0⁻（接近边界时），−1/gᵢ(x) → +∞——形成"无穷高的墙"。这就像一个隐形围栏，离得远时感觉不到，靠近时会被强力推回。

μᵢ 是小的正数，控制屏障的"强度"。μᵢ 越小，屏障越弱，最优解越接近真正的约束最优解。但 μᵢ 太小时数值问题会出现（接近边界时函数值剧烈变化）。

注意此方法只适用于**不等式约束**——因为需要一个"内部"来启动。等式约束没有"内部"，所以不适用。

### 第 ② 页：障碍函数法的几何直觉

**原文翻译：**

三维图示：水平面为决策变量空间，垂直轴为函数值。可行域由 gᵢ(x̂) < 0 定义，边界为 gᵢ(x̂) = 0。障碍项 −Σμᵢ/gᵢ(x) 在边界处趋于无穷大，形成"高墙"。

**讲解：**

这幅三维图是理解障碍函数法的关键。教授画出了：

1. **可行域**（水平面上的一个区域），边界由约束 gᵢ = 0 确定。
2. **目标函数** f(x) 的等高线（碗形曲面），其无约束最小值可能在可行域外。
3. **障碍项** −Σμᵢ/gᵢ(x) 在可行域边界处急剧上升至无穷大——图中表现为陡峭的"墙壁"。

合在一起看：原目标函数是一个"碗"，障碍项在可行域边界竖起"围墙"。梯度下降在这个"碗 + 围墙"的组合地形上运动，会自然地在可行域内部找到一个平衡点——既尽量降低 f，又不触碰围墙。

当 μᵢ 逐步减小时，围墙变矮但变薄，最优点逐步向真正的约束最优解靠近。这就是"内点法"名称的由来——解始终在可行域**内点**。

### 第 ③ 页：障碍函数法的完整算法

**原文翻译：**

算法三步：①选初始障碍参数、停止容差、缩减因子 β∈(0,1)、严格可行初始点；②用梯度下降从 xᵏ⁻¹ 出发求解障碍子问题得到 xᵏ；③若障碍项 ≤ ε 则停止，否则将 μᵢ 乘以 β（缩小）并继续。示例：同上讲的约束优化问题。

**讲解：**

与惩罚函数法的算法对比很有启发性：

| | 惩罚函数法 | 障碍函数法 |
|---|---|---|
| 参数变化 | θᵢ 递增（β > 1） | μᵢ 递减（β ∈ (0,1)） |
| 初始点要求 | 任意 | 必须严格可行 |
| 迭代点位置 | 可能在可行域外 | 始终在可行域内 |
| 适用范围 | 等式和不等式约束 | 仅不等式约束 |

**热启动（warm start）**策略在两种方法中都被使用：每次修改参数后，用上一轮的解作为新一轮梯度下降的初始点。这大大加速了收敛，因为参数变化不大时，新的最优解通常离旧的不远。

同一个例子 min (x₁−2)⁴ + (x₁−2x₂)² s.t. x₁²−x₂ ≤ 0 再次出现，学生可以比较惩罚函数法和障碍函数法在同一问题上的表现差异。

### 第 ④ 页：Taylor's Theorem（预备知识——小 o 记号）

**原文翻译：**

小 o 记号：o(t) 表示任何满足 lim_{t→0} g(t)/t = 0 的函数 g: ℝ → ℝ。例1：g(t) = t² 是 o(t) 吗？是——t²/t = t → 0。例2：g(t) = 2t 是 o(t) 吗？不是——2t/t = 2 ≠ 0。例3：g(t) = √t 是 o(t) 吗？不是——√t/t = 1/√t → ∞。

**讲解：**

从这里开始，课程进入数学基础准备阶段。小 o 记号是分析学中描述"比某个量增长更慢"的标准工具。

直觉理解：g(t) = o(t) 意味着"g(t) 比 t 趋向零的速度更快"。形象地说，当 t 很小时，g(t) "更小得多"。

三个例子覆盖了三种情况：
- t²：比 t 更快趋向零——是 o(t)。配图中 t² 曲线在原点附近"紧贴"横轴。
- 2t：与 t 同阶趋向零——不是 o(t)。它是 Θ(t)，即与 t 同速。
- √t：比 t 更慢趋向零——不是 o(t)。实际上 √t 比 t "大"（当 t 很小时 √t >> t）。

这个概念将在 Taylor 定理中发挥核心作用：f(y) ≈ f(x) + ∇f(x)ᵀ(y−x) 的误差是 o(‖y−x‖)——即线性近似的误差比距离本身还小。

### 第 ⑤ 页：Taylor's Theorem 的陈述

**原文翻译：**

聚焦无穷次可微函数 f: ℝⁿ → ℝ。Taylor 定理：f(y) = f(x) + ∇ᵀf(γx + (1−γ)y)·(y−x)，对某个 γ ∈ (0,1)。配图为一维情况的几何解释。

**讲解：**

这是多变量微积分中最重要的定理之一。一维版本就是**中值定理**：f(y) = f(x) + f'(c)(y−x)，其中 c 在 x 和 y 之间。多维版本用梯度 ∇f 替代导数 f'，中间点变成 γx + (1−γ)y——x 和 y 的凸组合。

配图展示了一维情况：在 x 和 y 之间存在某个点（红色标注），该点处的切线斜率恰好等于割线斜率 (f(y)−f(x))/(y−x)。这就是"中间值"的几何含义。

教授强调函数需要"无穷次可微"，这在实际应用中不是问题——大多数优化中遇到的函数（多项式、指数、对数等）都是无穷次可微的。

这个定理的重要性在于：它将函数值的差 f(y) − f(x) 与梯度联系起来，这正是分析梯度下降收敛性所需要的桥梁。

---

### 第 ⑥ 页：Taylor's Theorem 的推论

**原文翻译：**

回顾范数和内积。Taylor 定理的推论：f(y) = f(x) + ∇ᵀf(x)·(y−x) + o(‖y−x‖)。证明：将 Taylor 定理中的 ∇f(γx+(1−γ)y) 分解为 ∇f(x) + [∇f(γx+(1−γ)y)−∇f(x)]，利用内积的几何关系和连续性论证余项是 o(‖y−x‖)。

**讲解：**

这个推论是 Taylor 定理在优化理论中最常用的形式。它说：**函数在某点的值可以用该点的梯度做线性近似，误差比距离还小。**

证明的关键技巧是"加零"：

$$f(y) = f(x) + \nabla f(\text{中间点})^T(y-x)$$
$$= f(x) + \nabla f(x)^T(y-x) + \underbrace{[\nabla f(\text{中间点}) - \nabla f(x)]^T(y-x)}_{\text{需证为 } o(\|y-x\|)}$$

利用内积的性质 aᵀb = ‖a‖‖b‖cos θ，余项可以写成 ‖∇f(中间点)−∇f(x)‖·‖y−x‖·cos φ。要证这是 o(‖y−x‖)，只需证 ‖∇f(中间点)−∇f(x)‖ → 0 当 y → x 时。

---

### 第 ⑦ 页：Taylor's Theorem 推论的证明完成

**原文翻译：**

验证极限为零：当 ‖y−x‖ → 0 时，y−x → 0，因此 γx+(1−γ)y → x，由梯度的连续性知 ∇f(γx+(1−γ)y) → ∇f(x)，差值趋于零。证毕。

**讲解：**

证明的最后一步用到了梯度函数的**连续性**——这是由"无穷次可微"保证的。当 y → x 时，中间点 γx+(1−γ)y 也趋向 x（因为它是 x 和 y 的凸组合），所以 ∇f(中间点) → ∇f(x)。

这完成了 f(y) = f(x) + ∇f(x)ᵀ(y−x) + o(‖y−x‖) 的证明。这个公式的含义是：

1. **f(x) + ∇f(x)ᵀ(y−x)** 是 f 在 x 处的**一阶 Taylor 展开**（切平面近似）。
2. **o(‖y−x‖)** 是误差项，当 y 靠近 x 时，误差消失得比距离本身还快。

这为后续的梯度下降收敛分析提供了核心工具：在分析 f(xᵏ⁺¹) 与 f(xᵏ) 的关系时，Taylor 展开将 xᵏ⁺¹ = xᵏ − α∇f(xᵏ) 代入，得到 f(xᵏ⁺¹) ≈ f(xᵏ) − α‖∇f(xᵏ)‖²，从而证明每步函数值下降。
