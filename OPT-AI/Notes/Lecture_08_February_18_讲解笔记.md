## 逐页精讲

### 第 ① 页：Taylor's Theorem（复习）

**原文翻译：**

复习小 o 记号、范数、内积。Taylor 定理：f(y) = f(x) + ∇ᵀf(γx+(1−γ)y)·(y−x)，对某个 γ ∈ (0,1)。配图为中值定理的几何解释。

**讲解：**

教授在新一讲开头花了一整页复习 Taylor 定理，说明这个定理的重要性。本页也复习了基本的线性代数记号：xᵀx = Σxᵢ² = ‖x‖²，这个恒等式在后续推导中反复出现。

配图比上一讲更精细：不仅画出了函数曲线、割线（红色），还画出了函数在 x 和 y 之间的起伏——中间点 γx+(1−γ)y 处的切线恰好平行于割线。这就是"中值定理"的几何本质：无论函数在两点之间如何波动，总存在一个中间点的"瞬时变化率"等于"平均变化率"。

---

### 第 ② 页：Implication of Taylor's Theorem

**原文翻译：**

Taylor 定理的推论：f(y) = f(x) + ∇f(x)ᵀ(y−x) + o(‖y−x‖)。证明过程：将中间点梯度分解为 ∇f(x) + 差值项，利用内积和连续性论证余项。

**讲解：**

这一页重述了 Lecture 7 最后几页的证明，教授通过重复强化理解。推论的含义是：**f 在 x 附近可以用线性函数 f(x) + ∇f(x)ᵀ(y−x) 近似，误差是 o(‖y−x‖)**。

"o(‖y−x‖)"的直觉含义是"误差比距离还小"——当 y 非常接近 x 时，线性近似极其准确。这是所有基于梯度的方法有效的数学根基。

证明的核心手法"加零减零"是分析学中最常用的技巧之一：将 ∇f(中间点) 拆分为 ∇f(x) + [∇f(中间点) − ∇f(x)]，前者给出主项，后者给出误差项。

---

### 第 ③ 页：Taylor 推论的证明完成与应用

**原文翻译：**

完成极限验证。然后提出核心问题：为什么 Taylor 定理推论重要？在 x̂ 处 ∇f(x̂) ≠ 0，沿负梯度走小步。代入 Taylor 展开得到 f(x̂ − α∇f(x̂)) = f(x̂) − α‖∇f(x̂)‖² + o(α‖∇f(x̂)‖)。

**讲解：**

这是全讲最重要的推导。教授首先完成上一页的证明，然后立即转入**应用**——证明梯度下降每步确实能降低函数值。

关键计算：令 y = x̂ − α∇f(x̂)，代入 Taylor 推论：

$$f(\hat{x} - \alpha\nabla f(\hat{x})) = f(\hat{x}) + \nabla f(\hat{x})^T(-\alpha\nabla f(\hat{x})) + o(\alpha\|\nabla f(\hat{x})\|)$$

$$= f(\hat{x}) - \alpha\|\nabla f(\hat{x})\|^2 + o(\alpha\|\nabla f(\hat{x})\|)$$

核心项 −α‖∇f(x̂)‖² 是**严格负的**（当 ∇f(x̂) ≠ 0 时）。误差项 o(α‖∇f(x̂)‖) 在 α 小时可以被忽略。教授将表达式整理为 f(x̂) + α‖∇f(x̂)‖·(−‖∇f(x̂)‖ + 余项/α‖∇f‖) 的形式，方便后续取极限。

---

### 第 ④ 页：梯度下降函数值下降的严格证明

**原文翻译：**

选 α > 0 足够小使得误差项 ≤ ½‖∇f(x̂)‖。代入得 f(x̂ − α∇f(x̂)) ≤ f(x̂) − ½α‖∇f(x̂)‖² ≤ f(x̂)。

**讲解：**

证明的最后一步优雅地利用了 o(·) 记号的定义。既然 o(α‖∇f‖)/(α‖∇f‖) → 0 当 α → 0，我们总能找到足够小的 α 使得这个比值的绝对值不超过 ½‖∇f(x̂)‖——任何正数都可以，½‖∇f(x̂)‖ 的选择是为了让最终的不等式干净利落。

最终结论：

$$f(\hat{x} - \alpha\nabla f(\hat{x})) \leq f(\hat{x}) - \frac{1}{2}\alpha\|\nabla f(\hat{x})\|^2 < f(\hat{x})$$

这就是梯度下降有效性的**严格数学证明**。它告诉我们：只要梯度不为零，总存在足够小的步长使函数值严格下降。下降量至少为 ½α‖∇f‖²——步长越大、梯度越大，下降越多。

这个结论也解释了梯度下降的停止条件：当 ‖∇f(xᵏ)‖ 接近零时，我们接近了一个**驻点**（stationary point）。对于凸函数，驻点就是全局最小值；对于非凸函数，可能是局部最小值或鞍点。

### 第 ⑤ 页：Convex Functions

**原文翻译：**

凸函数定义：f: ℝⁿ → ℝ 是凸的，如果对所有 x, y ∈ ℝⁿ, λ ∈ [0,1] 满足 f(λx+(1−λ)y) ≤ λf(x)+(1−λ)f(y)。配图展示"弦在曲线上方"。

**讲解：**

凸函数的定义是整个优化理论的基石。几何含义极其直观："连接函数图像上任意两点的弦，始终在函数曲线的上方或与之重合。"

配图完美地展示了这个性质：蓝色曲线是凸函数 f，红色直线连接 (x, f(x)) 和 (y, f(y))。在凸组合点 λx+(1−λ)y 处，曲线上的值 f(λx+(1−λ)y) 低于红线上的值 λf(x)+(1−λ)f(y)。

这个定义虽然简洁，但蕴含了极其丰富的推论：没有"假谷底"、局部最优即全局最优、梯度下降全局收敛等。教授在前面几讲已经非正式地使用了这些结论，现在终于给出了严格定义。

重要特例：线性函数 f(x) = aᵀx + b 既是凸函数又是凹函数（等号成立）。二次函数 f(x) = xᵀAx 在 A 半正定时是凸函数。

### 第 ⑥ 页：Jensen's Inequality

**原文翻译：**

Jensen 不等式：若 f 凸，λ¹,...,λᵏ ≥ 0 且 Σλⁱ = 1，则 f(Σλⁱxⁱ) ≤ Σλⁱf(xⁱ)。证明通过归纳法——反复应用两点凸性定义。

**讲解：**

Jensen 不等式将凸性定义从"两点的凸组合"推广到"任意多点的凸组合"。直觉上说：凸函数在加权平均点处的值，不超过函数值的加权平均。

证明采用"剥洋葱"策略：先从 k 个点中分离出第一个点 x¹（权重 λ¹），对剩余点的凸组合应用凸性定义。然后对剩余点重复这个过程。每次应用凸性都是"两点"版本，所以不需要额外假设。

教授的证明中有很多中间步骤的代数细节——这是因为权重的归一化需要仔细处理。关键是每次剥离一个点后，剩余权重需要重新归一化：λⁱ/(1−λ¹) 保证了剩余权重之和为 1。

Jensen 不等式在概率论和统计中无处不在。如果 X 是随机变量、f 是凸函数，则 f(E[X]) ≤ E[f(X)]——"凸函数的期望不小于期望的凸函数"。

### 第 ⑦ 页：凸函数的梯度刻画

**原文翻译：**

梯度基础的凸性刻画。定理：f: ℝⁿ → ℝ 是凸的当且仅当 f(y) ≥ f(x) + ∇ᵀf(x)·(y−x)。配图展示切线在凸函数下方。

**讲解：**

这是凸函数最重要的等价刻画之一，其含义是：**凸函数在任何点的切线（一阶 Taylor 近似）都是全局下界。**

几何上：在凸函数图像上任取一点 x，画出切线。这条切线永远不会穿过函数曲线——它始终在曲线下方或与曲线相切。这与上一页的"弦在上方"形成互补：弦是上界，切线是下界。

数学上：不等式 f(y) ≥ f(x) + ∇f(x)ᵀ(y−x) 表明线性函数 L(y) = f(x) + ∇f(x)ᵀ(y−x) 是 f 的全局下界。在 y = x 时等号成立（L(x) = f(x)）。

这个性质在优化理论中极其重要：
1. 它给出了凸函数最小值的必要充分条件：∇f(x\*) = 0 ⟹ 对所有 y，f(y) ≥ f(x\*) + 0 = f(x\*)。
2. 它是推导梯度下降收敛率的关键工具——下一讲将用它来建立 O(1/k) 收敛率。
3. 它说明凸函数的一阶信息（梯度）已经包含了全局结构信息——这是凸函数"好优化"的根本原因。
