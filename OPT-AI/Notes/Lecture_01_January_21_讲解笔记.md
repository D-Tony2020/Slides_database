5. **逻辑回归**（第⑤-⑥页）：从分类问题出发，引入 sigmoid 函数作为预测函数，然后通过最大似然估计建立优化目标。

## 逐页精讲

### 第 ① 页：What do we mean by solving an optimization problem?

**原文翻译：**

解一个优化问题是什么意思？我们来设计一个具有最大可能体积的盒子，要求：盒子的底面是正方形；盒子的表面积不超过 500 cm²；盒子没有顶部。决策变量：x = 底面边长/宽度，y = 盒子高度。目标函数：x²y。约束：x² + 4xy ≤ 500，x ≥ 0，y ≥ 0。根据 Excel 求解，最优解为 (x̂, ŷ) = (12.91, 6.45)，最优体积为 1075.82。

**讲解：**

教授用一个非常具体的工程设计问题来开场，意图是让学生直观理解优化问题的基本结构。这个无顶正方形底盒子的设计问题虽然简单，却完美地包含了优化问题的三大要素：

**决策变量（Decision Variables）**：我们能控制什么？这里是底面边长 x 和高度 y。在 ML 中，决策变量通常是模型参数 w。

**目标函数（Objective Function）**：我们要最大化或最小化什么？这里是体积 V = x²y（底面积乘以高度）。在 ML 中，通常是最小化损失函数。

**约束条件（Constraints）**：有哪些限制？盒子没有顶，所以表面积 = 底面 x² + 四个侧面 4xy ≤ 500，加上非负性约束。在 ML 中，约束可能来自物理限制、预算、正则化等。

教授特意用 Excel 给出了数值解 (12.91, 6.45)，体积约 1075.82。这不是为了教 Excel，而是为下一页的"陷阱"做铺垫——Excel 作为通用优化工具在非线性问题上的脆弱性。红色标注的 Excel 数值结果暗示这是一个教学重点：优化工具给你的答案未必是全局最优。

### 第 ② 页：非线性优化问题的挑战

**原文翻译：**

这个优化问题是非线性的，因为目标函数在决策变量中是非线性的，或者约束的左端在决策变量中是非线性的，或者两者皆是。事实上，当我们求解非线性优化问题时，优化软件可能非常容易失败。例如，尝试用 Excel 求解前面的优化问题，初始猜测设为 x = 0，y = 0。

**讲解：**

这一页虽然简短但极其重要——它为整门课的存在意义"正名"。教授要传达的核心信息是：**非线性优化问题的求解是困难的，通用的"黑箱"软件不可靠**。

非线性性来自两个方面：目标函数 x²y 对 x 是二次的、对 y 是乘积关系（双线性）；约束 x² + 4xy ≤ 500 同样包含二次项和交叉项。这与线性规划（所有函数都是线性的）形成鲜明对比——线性规划有高效且可靠的算法（如单纯形法），而非线性优化则充满陷阱。

教授举的"初始猜测 x=0, y=0"的例子非常经典：(0,0) 是可行域的一个角点，梯度在该点可能为零或指向错误方向，导致 Excel 的求解器（通常是基于梯度的局部搜索）陷入这个"死点"无法脱身。这个例子生动说明了**初始点的选择对非线性优化至关重要**——同一个问题，换一个起点，结果可能天壤之别。

这就是为什么我们需要深入理解优化算法的原理，而不是盲目依赖软件——这正是这门课要教的。

### 第 ③ 页：Role of Optimization in Data Analysis, ML and AI

**原文翻译：**

优化在数据分析、机器学习和人工智能中的角色。ML/AI 应用中的典型数据集形如 {(xⱼ, yⱼ) : j=1,...,M}，其中 xⱼ ∈ ℝⁿ 是特征向量，yⱼ ∈ ℝ 是对应数据点 j 的标签。目标是找到一个由 w ∈ ℝᵖ 参数化的预测函数 ϕ(·, w): ℝⁿ → ℝ，使得 ϕ(xⱼ, w) ≈ yⱼ 在大部分数据点上成立。设 ℓ(xⱼ, yⱼ, w) 为使用参数为 w 的预测函数对第 j 个数据点的标签进行预测时产生的损失或"误差"。例如对于回归，ℓ(xⱼ, yⱼ, w) = (ϕ(xⱼ, w) − yⱼ)²。ML/AI 中的优化问题形如：min_{w∈ℝᵖ} (1/M)∑ℓ(xⱼ, yⱼ, w)。

**讲解：**

这一页建立了整门课的理论框架：**ML/AI 的训练过程本质上就是在解一个优化问题**。教授的叙述从具体到抽象，层层递进：

首先是**数据**：(xⱼ, yⱼ) 对，特征向量 xⱼ 是我们观测到的输入（如图片像素、用户行为等），标签 yⱼ 是我们想预测的输出（如类别、价格等）。M 个这样的数据对构成训练集。

然后是**模型**：预测函数 ϕ(x, w) 将输入映射到预测输出。关键在于它被参数 w 参数化——不同的 w 给出不同的预测器。找到好的 w 就是训练的目标。

接着是**损失函数**：ℓ(xⱼ, yⱼ, w) 量化了在参数 w 下对第 j 个样本的预测有多"差"。回归问题用平方误差 (ϕ(xⱼ, w) − yⱼ)² 是最自然的选择。

最后将三者组合成**经验风险最小化**（Empirical Risk Minimization, ERM）：对所有样本的损失取平均，然后关于 w 最小化。这就是 ML 训练的数学本质。这个公式看似简单，但当 ϕ 是深度神经网络时，它就变成了一个极其复杂的非线性优化问题——回扣上一页的主题。

### 第 ④ 页：Least Squares Regression

**原文翻译：**

最小二乘回归。我们将数据点 j 的标签预测为 wᵀ·xⱼ，其中 w ∈ ℝⁿ 是需要估计的参数向量。在这种情况下我们求解的优化问题形如：min_{w∈ℝⁿ} (1/M)∑(wᵀxⱼ − yⱼ)²。Ridge 回归：加上 λ‖w‖₂²（= ∑wᵢ²），有助于避免过拟合、保证鲁棒性、抵御离群值。Lasso：加上 λ‖w‖₁（= ∑|wᵢ|），有助于特征选择。

**讲解：**

这一页将上一页的抽象框架具体化到最简单的线性模型：ϕ(x, w) = wᵀx。预测就是特征向量与参数向量的内积，这是最基本的线性回归。此时损失函数变为 (wᵀxⱼ − yⱼ)²，整个优化问题是一个**二次函数最小化**——这是凸优化中最"友好"的一类问题，有解析解（正规方程）。

但教授立刻引入了两个关键的正则化变体：

**Ridge 回归（L2 正则化）**：在损失函数后加上 λ∑wᵢ²。L2 惩罚项将参数向量的模压小，不让任何分量变得过大。几何上，它将解限制在以原点为中心的球内。好处是：（1）防止过拟合——参数不会为了完美拟合噪声而变得极端；（2）提高鲁棒性——对数据中的离群值不那么敏感；（3）保证解的唯一性——即使特征矩阵不满秩也有唯一解。

**Lasso 回归（L1 正则化）**：加上 λ∑|wᵢ|。L1 惩罚项的关键性质是产生**稀疏解**——很多 wᵢ 会被精确压到零。这等价于自动特征选择：模型自行判断哪些特征重要（对应 wᵢ ≠ 0），哪些可以丢弃（对应 wᵢ = 0）。但 L1 范数不可微（在 wᵢ = 0 处），这给优化带来额外挑战。

从优化的角度看，Ridge 问题仍是光滑凸优化（二次可微），而 Lasso 是非光滑凸优化——这门课后续会分别讨论处理方法。

### 第 ⑤ 页：Logistic Regression

**原文翻译：**

逻辑回归。考虑一个两类分类问题，数据集 {(xⱼ, yⱼ) : j=1,...,M}，其中 xⱼ ∈ ℝⁿ，yⱼ ∈ {+1, −1}。我们的预测函数形如 ϕ(x, w) = 1/(1 + eʷᵀˣ)，其中 ϕ(x, w) 可解释为特征为 x 的数据点属于类别 +1 的概率。1 − ϕ(x, w) = eʷᵀˣ/(1 + eʷᵀˣ) 是特征向量为 x 的数据点属于类别 −1 的概率。

**讲解：**

从回归问题自然过渡到分类问题。分类与回归的关键区别在于标签 yⱼ 不再是连续实数，而是离散类别——这里考虑最简单的二分类 {+1, −1}。

逻辑回归的核心是 **sigmoid 函数**（也叫 logistic 函数）σ(z) = 1/(1 + e⁻ᶻ)。教授写的形式 ϕ(x, w) = 1/(1 + eʷᵀˣ) 实际上是 σ(−wᵀx)，注意指数上的符号。这个函数将任意实数 wᵀx 映射到 (0, 1) 区间，因此可以自然地解释为概率。

概率解释的优美之处在于：ϕ(x, w) 和 1 − ϕ(x, w) 自动互补，恰好分别给出属于两个类别的概率。当 wᵀx 很大的负值时（意味着特征强烈指向 +1 类），eʷᵀˣ ≈ 0，ϕ ≈ 1；反之当 wᵀx 很大的正值时，eʷᵀˣ 很大，ϕ ≈ 0。决策边界在 wᵀx = 0 处，此时 ϕ = 0.5——两类概率相等。

与线性回归不同，逻辑回归的优化问题（最大似然估计）没有解析解，必须用迭代优化算法求解——这又回扣了本课的核心主题。

### 第 ⑥ 页：最大似然估计拟合逻辑回归

**原文翻译：**

为了拟合逻辑回归模型，我们使用最大似然估计。表格列出每个数据点的编号、特征向量、标签及对应的似然值。数据集的似然为所有单点似然的乘积：L(w) = ∏ⱼ (对应每个数据点的似然项)。

**讲解：**

这一页用一张详细的表格展示了最大似然估计（MLE）在逻辑回归中的具体运作方式。

核心思想是：给定参数 w，每个数据点 (xⱼ, yⱼ) 出现的概率是多少？如果 yⱼ = +1，那么这个数据点被正确"解释"的概率就是 ϕ(xⱼ, w) = 1/(1 + eʷᵀˣʲ)；如果 yⱼ = −1，则概率是 eʷᵀˣʲ/(1 + eʷᵀˣʲ)。表格非常清晰地展示了这个对应关系。

**似然函数** L(w) 就是所有数据点的似然的乘积——这基于数据点独立同分布（i.i.d.）的假设。最大似然估计的目标是找到使 L(w) 最大的参数 w\*。

在实际计算中，我们通常取对数将乘积变为求和（对数似然），再取负号将最大化变为最小化：

min_w −log L(w) = −∑ⱼ log(对应似然项)

这就将统计估计问题转化为了一个优化问题。这个优化问题是凸的（负对数似然是凸函数），但没有闭式解——必须用本课将要学习的迭代算法（如梯度下降）来求解。

教授用这个例子完美地闭合了第一讲的逻辑：从"什么是优化" → "ML 的核心是优化" → "具体的 ML 问题（回归和分类）如何转化为优化问题"。后续课程将聚焦于**如何高效地求解这些优化问题**。
